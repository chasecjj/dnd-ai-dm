Before the Session

Session prep is optional but powerful. Use the War Room channel to plan ahead.

Starting the Session

  1. Open the console: /console
  2. Click ▶ Start Session
  3. Queue mode auto-enables, session log created, “Previously on...” recap posted

Step-by-Step Turn Cycle

  1. Players type actions in Game Table (each gets ⏳)
  2. DM clicks 🔍 Analyze (Rules Lawyer checks what rolls are needed)
  3. Players roll dice when prompted
  4. DM manages queue (Annotate, Monster Roll, DM Event)
  5. DM clicks ▶ Resolve Turn (AI narrates the round)
  6. Repeat

Two Modes

Queue Mode
  • Player messages queue up with ⏳
  • Nothing happens until DM reviews
  • Full control over turn order and narrative
  • Best for: Combat, important story beats
Auto Mode
  • Player messages immediately trigger AI
  • Narrative posted within seconds
  • No DM review
  • Best for: Free exploration, casual RP

Ending the Session

  1. Click ⏹ End Session
  2. AI generates session summary, saves to vault
  3. Session number incremented, queue disabled

Secrets of the Craft

Seven tips from seasoned Dungeon Masters

  • Analyze early. Click Analyze as soon as actions arrive. This gives players time to roll while you review.
  • Batch intelligently. Include related actions in the same resolve. The AI handles multi-character narration well.
  • Use annotations freely. The AI reads your DM notes and adjusts its storytelling. "Make this dramatic" or "This should feel eerie" work great.
  • Monster rolls before Resolve. Roll monster attacks/damage before clicking Resolve so the AI has all the information for narration.
  • Auto Mode for downtime. Switch to Auto Mode for tavern RP and exploration, then back to Queue Mode when combat starts.
  • The console thread persists. You can close and reopen it with /console without losing state.
  • If the pipeline crashes. Actions are automatically restored to the queue. Check Moderator Log for errors, then try resolving again.
