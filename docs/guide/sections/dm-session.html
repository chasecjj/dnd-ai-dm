The Session Lifecycle

From preparation to the final curtain

Before the Session

Session prep is optional but powerful. Use the War Room channel to plan ahead.

Command What It Does
!prep Generate a prep outline for your next session based on campaign state
!brainstorm Collaborative AI brainstorming for encounters, NPCs, or plot hooks
!plan Create a structured session plan with objectives and contingencies
!build Generate a Foundry VTT scene with tokens and lighting
!scene Set up or switch the active Foundry scene

Starting the Session

  1. Open the console: /console
  2. Click ▶ Start Session
  3. Queue mode auto-enables, session log created, “Previously on...” recap posted

The Turn Cycle

Players Type Actions

Resolve
Narrative Posted

Step-by-Step Turn Cycle

  1. Players type actions in Game Table (each gets ⏳)
  2. DM clicks 🔍 Analyze (Rules Lawyer checks what rolls are needed)
  3. Players roll dice when prompted
  4. DM manages queue (Annotate, Monster Roll, DM Event)
  5. DM clicks ▶ Resolve Turn (AI narrates the round)
  6. Repeat

Two Modes

Queue Mode
  • Player messages queue up with ⏳
  • Nothing happens until DM reviews
  • Full control over turn order and narrative
  • Best for: Combat, important story beats
Auto Mode
  • Player messages immediately trigger AI
  • Narrative posted within seconds
  • No DM review
  • Best for: Free exploration, casual RP

Ending the Session

  1. Click ⏹ End Session
  2. AI generates session summary, saves to vault
  3. Session number incremented, queue disabled
